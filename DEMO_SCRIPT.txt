"This is one of our four innovative features — Predictive Performance Alerts,

The system trains a small neural network for each staff member using their
historical KPI scores, and predicts what their performance will look like

This threshold bar here shows exactly how the system classifies the predictions —

  QUICK REFERENCE — KEY POINTS TO MENTION
================================================================================

  SYSTEM:
  - Supervisor-only system (no staff login)
  - 13 staff, 21 KPIs, data from 2022 to 2026
  - Built with PHP, MySQL, Bootstrap 5, Chart.js, TensorFlow.js

  INNOVATIVE FEATURES:
  1. Predictive Alerts    — TF.js neural network, runs in browser
  2. Training Recommendations — skill gap matching, 6 programs
  3. Anomaly Detection    — spike/drop/decline detection + narrative
  4. Peer Comparison      — similarity scoring, specific staff option

  FORMULA (if asked):
  - weighted_score = (score / 5) × (weight / 100)
  - overall % = sum of all 21 weighted scores × 100

  TENSORFLOW.JS (if asked):
  - Dense neural network: Input → Dense(8) → Dense(4) → Dense(1)
  - Trains in the browser using historical yearly scores
  - Predicts next year's score, classifies into 4 risk levels
  - No Python or server-side ML needed
